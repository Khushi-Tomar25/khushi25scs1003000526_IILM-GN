Bank Management System
This C++ program simulates a simple bank management system that allows users to manage account details, deposit money, and withdraw money.

->Features
Create account with details (name, account type, account number, balance)
Deposit money
Withdraw money
Display updated balance after each transaction

->Concepts Used
Classes & Objects
Constructors
Member Functions

-> Working
User enters account details
System allows deposit and updates balance
System allows withdrawal and updates balance accordingly

->Example Output
Name of Account Holder: Lisa
Account Type: Saving
Account Number: 4567089
Account Balance: 6500.8

Deposit Amount: 500

Withdraw Amount: 200
