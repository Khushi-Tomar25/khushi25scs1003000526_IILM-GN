📌 Scam Link Analyzer

A simple and fast web-based tool to detect whether a URL is Safe, Suspicious, or a Scam.

🚀 Features
    🔍 Analyze any URL instantly
    ⚡ Fast and lightweight system
    🎯 Simple output: Safe / Suspicious / Scam
    🌐 Works across devices (frontend + backend connection)
    🔗 API-based architecture (easy to scale)
    🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Communication: REST API (JSON)

⚙️ How It Works
~ User enters a URL in the frontend
~ Frontend sends the URL to backend using a POST request
~ Backend analyzes the URL using rule-based logic:
    Suspicious keywords
    URL length
    Domain patterns
Backend returns result:
    ✅ Safe
    ⚠️ Suspicious
    ❌ Scam
