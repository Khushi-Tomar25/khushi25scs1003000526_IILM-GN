## Number Operations using Friend Function and Operator Overloading
This program performs basic operations on two numbers using concepts of friend functions and operator overloading.

  -> Features
- Takes two numbers as input
- Calculates their sum
- Calculates their subtraction
- Multiplies the obtained results

  -> Concepts Used
- Classes and Objects
- Friend Function
- Operator Overloading

  -> Example Output
Fixed values:
a = 1, b = 1

Parametric values:
a = 2, b = 4

Results:
Added value: 3 5  
Subtracted value: -1 -3  
Multiplied value: -3 -15

## String Check using Operator Overloading
This program stores a message in a string and checks whether the user has entered a message or not using operator overloading.

  -> Concepts Used
- Classes and Objects
- Constructors
- Operator Overloading
- String Handling

  -> Working
- If no message is entered, the program displays: "No message entered"
- Otherwise, it displays: "User entered a message"

  -> Example Output

  Case 1: No Message Entered
Default message: Hi  
User Message:  
Output: No message entered  

  Case 2: Message Entered
Default message: Hi  
User Message: Hello world  
Output: User entered a message
  
## Student Attendance Eligibility using Operator Overloading
This C++ program calculates a student's attendance percentage and determines whether the student is eligible to appear for the exam using operator overloading.

  ->Features
- Calculates attendance percentage
- Determines exam eligibility (Allowed / Debarred)
- Uses member operator overloading (`!`)
- Demonstrates Object-Oriented Programming concepts

  -> Concepts Used
- Classes & Objects
- Constructor
- Operator Overloading

  -> Working
- The program calculates attendance percentage based on input
- If attendance is below the required limit, student is **Debarred**
- Otherwise, student is **Allowed** to sit in the exam

  ->Example Output
Total Classes: 100  
Attended Classes: 70  
Attendance Percentage: 70%  
Status: Allowed

## Bank Management System

   ->Features
- Create account with details (name, account type, account number, balance)
- Deposit money
- Withdraw money
- Display updated balance after each transaction

   ->Concepts Used
- Classes & Objects
- Constructors
- Member Functions

  -> Working
- User enters account details
- System allows deposit and updates balance
- System allows withdrawal and updates balance accordingly
